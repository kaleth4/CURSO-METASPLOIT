## вҢЁпёҸ Comandos BГЎsicos
 
```bash

msf > search cve:2009-3843
 
# Cargar mГіdulo por nombre o Гӯndice
```

## рҹ“ҡ Conclusiones Clave
 
### Tarea 1 вҖ” Entorno Virtual
- Metasploit existe en versiГіn **community** (gratuita, preinstalada en Kali) y **Pro** (licenciada)
- Las VMs permiten crear entornos atacante-vГӯctima de forma segura y rГЎpida
 
### Tarea 2 вҖ” MГіdulos
- Metasploit se opera desde `msfconsole`
- Los mГіdulos son scripts especializados: `auxiliary`, `exploit`, `payload`, `encoder`, `post`
- Combinados, pueden ser extremadamente potentes
 
### Tarea 3 вҖ” Comandos
- `help` вҶ’ guГӯa de comandos disponibles
- `search` вҶ’ buscar exploits por nombre o CVE
- `use` вҶ’ cargar mГіdulo
- `options` вҶ’ ver parГЎmetros requeridos
- `set` вҶ’ asignar valores
- `run` / `exploit` вҶ’ ejecutar
 
### Tarea 4 вҖ” PrГЎctica EternalBlue
- Vulnerabilidad activa desde **2017** con exploit disponible en Metasploit
- Proceso: escaneo TCP вҶ’ validar puerto 445 вҶ’ scanner SMB вҶ’ exploit
 
### Tarea 5 вҖ” Payloads
- **Singles**: auto-contenidos, pesados pero efectivos
- **Stagers**: conexiГіn ligera hacia el atacante
- **Stages**: descargados por stagers вҖ” Meterpreter ofrece acceso avanzado
 
### Tarea 6 вҖ” ExplotaciГіn
- `check` valida vulnerabilidad antes de ejecutar
- Una explotaciГіn exitosa entrega sesiГіn `AUTHORITY\SYSTEM`
- Posibles impactos: robo de datos, modificaciГіn de archivos, apagado remoto
